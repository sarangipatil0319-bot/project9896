# TechStore - React E-Commerce App

A beautiful electronics e-commerce homepage built with React.js.

## 🛍️ Features
- Responsive Navbar
- Hero Section with banner image
- Product Categories
- Featured Product Cards with images
- Customer Testimonials
- Footer

## 🧩 Components
- `Navbar.jsx` - Navigation bar
- `Hero.jsx` - Hero banner section
- `Categories.jsx` - Product categories
- `ProductCards.jsx` - Featured products
- `Testimonials.jsx` - Customer reviews
- `Footer.jsx` - Page footer

## 🚀 How to Run
```bash
cd stylehub
npm install
npm start
